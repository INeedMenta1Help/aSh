aSh getting Started:
	aSh was designed for ease of use to read write and clean files with ease
	by using the 3 simple commands, aSh can completely wipe any file of all its data, write new data,
	and show the file completely, all within the same Shell

Commands:
	clean: file.txt
		completely wipes all data from the file and replaces it with: "Cleaned By aSh Version: *ver"
	
	fileread: file.txt
		Shows any file within the shell:

	filewrite: file.txt
		allows the user to write new lines onto the end of the program


basic commands:
	cls
		clears the current screen

	println: xxx
		Prints xxx to the screen
	
	exit
		exits the "aSh" program
